Query Power On Type

The reply to the SMC command 0x01 will contain some information about the event which caused the X360 to power up.
Message: 01
Reply: 01 WW 00 XX YY ZZ
Data Source Meaning
WW @63h Type of power up Event
XX @3fh ??
YY.0 @2bh.1 ??
ZZ @3eh ??
